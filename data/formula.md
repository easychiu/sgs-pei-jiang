## 1. 完整傷害公式

### 1.1 核心公式（兵刃 / 謀略均適用）

```
最終傷害 = 基礎傷害 × 兵種克制係數 × 士氣係數 × 隨機浮動 × 功能性增減傷

基礎傷害 = ((攻擊屬性 - 防禦屬性) / 150 + 1) × (帶兵數 / 20) × 戰法傷害率
```

展開版（逐項）：
```
最終傷害
  = [ (Atk - Def) / 150 + 1 ]
  × [ 兵力 / 20 ]
  × 戰法傷害率            ← 普攻 = 1.0（即100%）
  × 等級壓制係數          ← 見 §6
  × 兵種克制係數          ← 見 §2
  × 士氣係數              ← 見 §3
  × 隨機浮動              ← 見 §6
  × (1 + 暴擊加成)        ← 見 §5
  × 功能性增傷倍率
  × (1 - 功能性減傷倍率)
```

#### 屬性差對應傷害貢獻
- 每 1 點「攻擊屬性 - 防禦屬性」增加 ≈ 0.67% 傷害（由公式 1/150 ≈ 0.00667 推導）
- 攻守相等時（差=0），係數 = 1.0，不加也不減
- 可信度：**推估（由公式直接計算）**

#### 兵刃 vs 謀略使用的屬性
| 傷害類型 | 攻擊屬性 | 防禦屬性 |
|---------|---------|---------|
| 兵刃傷害 | 武力 | 統率（統帥） |
| 謀略傷害 | 智力 | 智力（敵方） |

來源：https://zhuanlan.zhihu.com/p/439300738、https://www.3h3.com/news/206614.html
可信度：**玩家實測（多來源一致）**

## 2. 兵種克制係數

### 2.1 克制關係
- 騎兵 → 剋 盾兵（楯兵）
- 盾兵 → 剋 弓兵
- 弓兵 → 剋 槍兵（長槍兵）
- 槍兵 → 剋 騎兵
- 器械（攻城）← 所有兵種均剋之

### 2.2 克制係數數值
| 狀態 | 傷害倍率 | 效果 |
|------|---------|------|
| 克制方（攻擊） | ×1.15 | 造成傷害 +15% |
| 被克制方（攻擊） | ×0.85 | 造成傷害 -15% |
| 無克制關係 | ×1.00 | 無加減 |

備註：兩方合計差距 = 1.15 / 0.85 ≈ 1.35，即「剋制」與「被剋制」相差約 35% 傷害。

來源：https://www.gamersky.com/handbooksy/202202/1458458.shtml、https://m.7724.com/sgzzlb/news/36512.html
可信度：**玩家實測（多來源一致）**

## 3. 士氣機制

### 3.1 初始值與上限
| 項目 | 數值 |
|------|------|
| 基礎出征士氣上限 | 100 |
| 升級士氣壇後上限 | 120（主城大殿 10 級後可建）|
| 建議戰鬥最低士氣 | 80（低於此值影響顯著）|

### 3.2 士氣消耗與恢復
- 行軍每格：-1 士氣
- 停留恢復：每 2 分鐘 +1 士氣（即 60 分鐘 +30 士氣）

### 3.3 士氣對傷害的乘數
公式（推導自實測對照表）：
```
士氣係數 = 士氣值 × 0.7% + 30%
         = 0.007 × 士氣 + 0.30
```
或等價寫為：
```
傷害保留率 = 30% + 0.7% × 士氣值
```

| 士氣值 | 傷害保留率 | 傷害削減 |
|--------|-----------|---------|
| 100    | 100%      | 0%      |
| 80     | 86%       | -14%    |
| 60     | 72%       | -28%（*） |
| 50     | 65%       | -35%    |
| 0      | 30%       | -70%    |

（*）注：部分來源對 60 士氣的削減值表述不一致（有稱 -35%，有稱 -28%）。根據線性公式推算，60 士氣應為 -28%；「60 士氣降低 35%」可能是把 50 士氣的數據誤填。
**衝突**：暫以線性公式為準，以 50 士氣對應 -35% 為錨點。

來源：https://syzs.qq.com/blog/news/20201218A03PFQ00、https://www.gamersky.com/handbooksy/202008/1310333.shtml
可信度：**玩家推測 / 部分實測**

## 4. 戰法發動率與準備回合

### 4.1 戰法類型與發動率
| 戰法類型 | 發動方式 | 概率 |
|---------|---------|------|
| 被動戰法 | 準備回合 100% 自動發動 | 100% |
| 指揮戰法 | 準備回合 100% 自動發動，武將死後仍生效 | 100% |
| 陣法戰法 | 100% 發動 | 100% |
| 主動戰法 | 每回合按概率判定，「計窮」狀態下無法發動 | 20%~55% |
| 突擊戰法 | 普攻後觸發判定，「繳械」狀態下無法發動 | 25%~55% |

### 4.2 主動戰法發動率區間（實測代表值）
| 概率區間 | 代表戰法（舉例） |
|---------|---------------|
| 20~25%  | 挫志、傾國傾城 |
| 30%     | 所向披靡（需準備1回合）|
| 35%     | 破陣摧堅、落鳳、大戟士、骨戾無仁 |
| 40%     | 橫掃千軍、百騎劫營 |
| 45%     | 風助火勢、臥薪嚐膽 |
| 50%     | 四面楚歌、唇槍舌戰 |
| 55%+    | 夺魂挾魄 |

來源：https://yosheng.github.io/sgzzlb-book/tactics-collection.html、https://www.gamersky.com/handbooksy/202004/1281487.shtml
可信度：**玩家整理（戰法面板數值）**

### 4.3 準備回合機制
- **定義**：帶「準備 N 回合」的主動戰法，發動前需先進入「準備狀態」。
- **最早生效**：第 2 回合（第 1 回合為準備，不輸出傷害）。
- **準備中行動**：準備回合期間不輸出普通攻擊（占用武將行動）。
- **期望折算**：因需要額外準備回合，期望係數折半 = 0.5（相較瞬發戰法的係數 1.0）。
- **公式**：在 8 回合戰鬥中，期望發動次數 = 面板概率 × 8 × 係數（瞬發=1，準備1回合=0.5）。

來源：https://www.gamersky.com/handbooksy/202004/1281487.shtml、https://bilibili.com/read/cv16728723
可信度：**玩家推估（理論計算）**

## 5. 暴擊（會心）機制

### 5.1 觸發方式
- **戰法觸發**：部分戰法說明直接帶有「N% 會心」概率。
- **武將被動**：甘寧「錦帆」自帶 50% 會心概率。
- **狀態疊加**：會心概率可由縁分、兵書、武器、陣法等疊加。
- **典型機制**：「會心」= 60% 外層概率 × 20% 內層概率（雙重判定）。

### 5.2 暴擊倍率
| 武將 | 會心傷害加成 | 換算倍率 |
|------|------------|---------|
| 一般武將 | +100% 傷害 | 2.0× |
| 甘寧（主將） | +120% 傷害 | 2.2× |
| 甘寧隊伍友軍 | +110% 傷害 | 2.1× |

來源：https://m.ali213.net/news/gl2102/572767.html、https://game.xiaomi.com/viewpoint/1279764826_1654053581697_16
可信度：**玩家實測**

## 6. 隨機浮動與等級壓制

### 6.1 隨機浮動範圍
- 傷害在計算結果上下浮動 **±3.3%**（即 96.7%～103.3%）。
- 部分來源表述為「96%～104%」（差距略大，為取整說法）。
- **分布方式**：非連續均勻，而是**分 7 個離散檔位**分布。

**衝突**：
- 來源 A（7724）：浮動約 3.3%，7 檔離散
- 來源 B（搜索摘要）：96%～104%（即 ±4%）

整體偏差不大，模擬建議使用 96%～104% 均勻分布於 7 檔（每檔間距約 1.3%）。

來源：http://www.7724.com/sgzzlbyx/news/150393.html
可信度：**玩家實測（推算）**

### 6.2 等級壓制係數
- 20 級以前：等級係數 = 1.0
- 20 級以後：每升 1 級 +0.02，滿級（50 級）係數 = 1.6
- **等級壓制效果**：高等級打低等級有 0%～9.2% 的額外傷害加成，差 10 級約額外 5.5% 增減傷。
- 最高等級壓制傷害差：約 20%（滿紅 vs 低等級）。

公式：`等級係數 = 1 + max(0, 等級 - 20) × 0.02`（上限係數 1.6）

來源：https://zhuanlan.zhihu.com/p/474417701
可信度：**玩家推測（存在爭議，非官方確認）**

## 7. 屬性換算：面板屬性 × 兵種適性

### 7.1 兵種適性等級對應百分比
| 適性等級 | 屬性發揮百分比 |
|---------|-------------|
| S | 120% |
| A | 100% |
| B | 85% |
| C | 70% |

### 7.2 完整屬性計算公式
```
最終面板屬性
  = (基礎屬性 + 裝備屬性)
  × (兵種適性% + 仙人加成%)   ← 仙人加成 30%（僅左慈/于吉/華佗）
  × (1 + 國家協力加成%)       ← 滿級協力 = +10%
  + 兵戰建築加成              ← +20 點
  + 弓兵營加成                ← +10 點全屬性
  + 縁分 / 戰法固定加值

基礎屬性 = 1 級屬性 + 成長屬性 × 等級 + 加點屬性
```

示例（50 級關羽，枪兵 S 适性）：
- 基礎武力 266.28
- × 枪兵 S 120% = 319.54
- × (1 + 10% 協力) = 351.49
- + 建築固定值 20 + 縁分 10 = 381.49（最終面板）

來源：https://news.17173.com/z/sgzzlb/content/07212020/155523813.shtml、https://www.gamersky.com/handbooksy/201912/1247763.shtml
可信度：**玩家整理（已多方驗證）**
