# 三國志戰略版 · 配將模擬器

**線上使用 → https://easychiu.github.io/sgs-pei-jiang/**

台服武將配將評分 + 逐回合模擬對戰 + 自動推薦,193 武將卡片圖鑑。
純靜態網站,瀏覽器內運算,無後端。**非官方社群工具,免費使用。**

## 功能

- **配將模擬**：選兩隊(各最多 3 將)+ 兵種 + 兵書/裝備 → 模擬 3000 場出勝率
- **自動推薦**：依勢力篩選,列出評分最高的陣容(含建議兵種)
- **武將圖鑑**：193 張華麗中國風卡片,點開看六維、五兵種適性、戰法白話說明

## 戰鬥模型

傷害公式(社群拆解,見 `data/formula.md`):
```
傷害 = ((攻-防)/150 + 1) × (兵力/20) × 戰法係數
       × 兵種克制 × 士氣係數 × 增傷 × (1-減傷) × random(0.96~1.04)
```
- **兵種由「隊伍」決定**(玩家選),各武將只有對該兵種的**適性 S/A/B/C/D** 決定屬性發揮(×120/100/85/70/55%)
- **克制是隊伍兵種 vs 隊伍兵種**:騎>盾>弓>槍>騎(器械全被克),克制 ×1.15 / 被克 ×0.85
- 逐回合、速度排序;主動戰法依發動率、突擊普攻後觸發、指揮/被動穩定
- **戰法 15 原語**:增傷/減傷/控制/治療/屬性/持續傷害(DoT)/結算傷害(猛毒)/連擊/傷害轉移(代承)/疊加/衰減/武智互換/看破/反擊
- **兵書**(47):裝備為自身被動 · **緣分**(20):隊伍湊齊觸發全隊加成 · **裝備**(35):自身被動

## 已知近似(精度天花板)

引擎是**趨勢正確的近似**,非逐幀還原:
- 條件/多模式戰法(「敵出主動時」「第N回合起」「奇偶切換」)無法判條件,指揮/被動傷害用觸發折扣近似(`sgz.py` 頂部 `CMD_TRIGGER`/`PASSIVE_TRIGGER` 旋鈕)
- 部分戰法數值經考據修正,帶 `_src`/`_real`/`_conf` 註記;`_conf=med` 以官方面板為準
- 武將取 Lv50、雙方同級(等級壓制略過);士氣固定滿
- 改資料不必動引擎,效果原語都在 JSON

## 開發

網站(`docs/`)純 vanilla JS,無 build step;`docs/engine.js` 是 `sgz.py` 的 JS 移植,兩版結果一致。

```bash
python sgz.py                              # demo: 推薦 + 一場模擬
python sgz.py sim "呂布,趙雲,關羽" "諸葛亮,周瑜,司馬懿" [兵種A 兵種B]
python sgz.py rec [勢力]                   # 配將推薦
python sgz.py info 呂布                     # 武將面板 + 五兵種適性 + 戰法解析
python sgz.py test                         # 自檢
```
Windows 主控台中文亂碼加 `PYTHONUTF8=1`。本地預覽網站:`python -m http.server --directory docs`。
